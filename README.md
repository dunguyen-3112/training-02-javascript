### 5. Getting started

- Node version: 16.17.1

- Clone repository

```
    git clone https://github.com/dunguyen-3112/training-02-javascript.git
```
- Checkout branch: Develop

```
git checkout develop
```

- Install package dependencies

```
npm install
```

- Run server

```
npm run server
```

- Run website

```
npm start
```
